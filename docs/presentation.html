The Problem

🎓

Graduates Are Lost

Students and fresh graduates lack structured career guidance. They don't know which skills to learn, what roles to target, or how to plan their journey.

📄

Resumes Are Weak

Most resumes fail ATS screening. Candidates don't know their skill gaps, miss keywords, and lack quantified achievements.

🗺️

No Clear Roadmap

Generic "learn Python" advice doesn't work. Students need personalized, week-by-week plans with resources, projects, and milestones.

💰

Salary & Market Blind Spots

Students can't assess their market readiness, expected salary, or which companies are hiring for their target role.

Career Intelligence

LLM Path

When OpenAI API key is configured, agents use GPT-4 for intelligent analysis and generation.

Fallback Path

Without an API key, agents use regex extraction, heuristic scoring, and template-based generation.

Export Pipeline

Data → Markdown → HTML (Jinja2) → PDF (xhtml2pdf). Professional outputs every time.
